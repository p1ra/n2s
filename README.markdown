Nasm 2 Shellcode                                                                                                          
================
                                                                                                                           
Description
-----------
                                                                                                                            
Simple wrapper to compile a nasm assembly files and print the code in escaped format.                                       
                                                                                                                            
Limitations
-----------                                                                                                                
* Currently only supports 32bit mode.
